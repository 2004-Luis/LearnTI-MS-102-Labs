# Exercício 2 - Atribuição de Licença

## Objetivo
Atribuir licença Microsoft 365 Business Premium ao utilizador trainee-LAB-A04.

## Passos
1. Acedi ao Microsoft 365 Admin Center.
2. Naveguei para Users > Active Users.
3. Selecionei o utilizador trainee-LAB-A04.
4. Atribuí a licença Microsoft 365 Business Premium.

## Resultado
O utilizador possui agora acesso aos serviços do Microsoft 365.
