## Item 9 – Criação de Grupo Microsoft 365

Foi criado um grupo Microsoft 365 chamado M365-PROJECT-LAB-A04
no Microsoft 365 Admin Center.

Este tipo de grupo cria automaticamente vários recursos
de colaboração no ecossistema Microsoft 365, incluindo:

- Caixa de correio no Exchange Online
- Site de equipa no SharePoint
- Integração com Microsoft Teams

Passos realizados:

1. Acedi ao Microsoft 365 Admin Center.
2. Naveguei até à secção "Grupos".
3. Criei um novo grupo do tipo Microsoft 365.
4. Defini o nome M365-PROJECT-LAB-A04.
5. Configurei o utilizador trainee-LAB-A04 como Owner do grupo.

Resultado:
O grupo foi criado com sucesso e está pronto
para suportar colaboração entre membros da equipa.
