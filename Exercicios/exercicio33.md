## 33 – Criação de Team baseado em Grupo

Neste exercício foi criada uma equipa no Microsoft Teams
baseada no grupo Microsoft 365 M365-SALES-LAB-A04.

Passos realizados:

1. Acedi ao Microsoft 365 Admin Center.
2. Naveguei até à secção Equipas e Grupos.
3. Localizei o grupo M365-SALES-LAB-A04.
4. Abri as configurações do grupo.
5. Selecionei a opção para criar uma equipa no Microsoft Teams.
6. Confirmei a criação.

Resultado:

A equipa foi criada com sucesso e encontra-se
disponível no Microsoft Teams para colaboração
entre os membros do grupo.
